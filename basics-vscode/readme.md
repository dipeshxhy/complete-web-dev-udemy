# shortcuts

- cmd + <- = move at beginning
- cmd + -> = move at end
- opt + <- = move at single word left
- opt + -> = move at single word right
- opt+ click = to get multiple lines
- opt + shift + up/down arrow = to copy line
- ctrl + x = to delete or cut the line
- opt + up/down arrow = to move line of code
- cmd + D = to select duplicate code

## Terminal shortcut

- cmd + ` = to open integrated terminal
- ctrl + delete = to delete entrire lines

- cmd + arrow-left / arrow-right = to move start or end of line
- opt + arrow-left / arrow-right = to move one word
- ctrl + K = to delete entire on right
- ctrl + D = to delete single character on right
- opt + delete = one word delete backwards
